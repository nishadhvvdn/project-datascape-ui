Summary Map
The summary map section is used to get a graphical overview of the current state of the system as well as perform basic meter functions like on-demand reads.
Outage Map
The outage map section is used to get a visual representation of current outages as well as view historical ones.
Non-Technical Loss Map
The non-technical loss map identifies any areas that are suspected of theft as well as the amount of expected theft.
Network Coverage Map
The network coverage map provides a general idea of the Wi-Fi coverage of the network and reports on the response rates of devices.
