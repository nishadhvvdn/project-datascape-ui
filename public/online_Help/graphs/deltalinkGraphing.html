Graphing Tools

Each graph has several different tools available to you. These tools are contained in a column to the right of the graph. From the top, down, the available functions are:

  • 1. Circuit ID: This dropdown can be used to select what DTCs are used to plot the graph
  • 2. Transformer ID: This dropdown can be used to select what Transformers are used to plot the graph
  • 3. Meter ID: This dropdown can be used to select what Meters are used to plot the graph
  • 4. Date: The date dropdown allows you to create a custom timeframe to plot the graph on
  • 5. View Table: The view table button creates a table that shows all the data used to plot the graph

Beyond these functions, the graphing page also supports two other pieces of functionality you may take advantage of. These include hi-lighting a single line on a multi-line graph–which can be done by clicking appropriate line in the legend–and pulling up more detail on a single point in the graph by scrolling directly over it.

View Table

On every graphing page there is a button called "View Table". When clicked, you are taken to a table view of all of the data used to create the graph. This table can be directly modified from that same page (i.e. the date and devices shown can be changed in the same way you would on the graphing page).

Data Consumption

On every graphing page there is a button called "Data Consumption". When clicked, you are taken to a graph view of all of the data used to create the graph. Device will send data consumed information to the cloud periodically. Data will get inserted to the mongodb. Datamart data consumption function will run to migrate data from MySQL to mongodb. Once the data is migrated graphs will be shown on DataSCAPE™ for DeltaLINK™.

Data Rate

Under DeltaLINK™ Data Consumption page, we have Data Rate Button when user click on it will take it to Data Rate Page. Here user need to select the appropriate Circuit ID and HS/HH ID then click on procced button if the device is online it will fetch the Data Download and Upload rate.
