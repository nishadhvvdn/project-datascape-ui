Graphing Tools

Each graph has several different tools available to you. These tools are contained in a column to the right of the graph. From the top, down, the available functions are:

  • 1. Graph Type: Here you can select what type of graph to display from a dropdown (this is called outage type on the outage graphing page)
  • 2. DTC ID: This dropdown can be used to select what DTCs are used to plot the graph
  • 3. Date: The date dropdown allows you to create a custom timeframe to plot the graph on
  • 4. View Table: The view table button creates a table that shows all the data used to plot the graph

Beyond these functions, the graphing page also supports two other pieces of functionality you may take advantage of. These include hi-lighting a single line on a multi-line graph–which can be done by clicking appropriate line in the legend–and pulling up more detail on a single point in the graph by scrolling directly over it.

View Table

On every graphing page there is a button called "View Table". When clicked, you are taken to a table view of all of the data used to create the graph. This table can be directly modified from that same page (i.e. the date and devices shown can be changed in the same way you would on the graphing page).

kWh Delivered

The kWh delivered graph shows two lines. The first plots the average kw received by all the devices in the selected transformers over time and the second plots the avg ambient temperature around the devices.

Data Consumption

On every graphing page there is a button called "Data Consumption". When clicked, you are taken to a graph view of all of the data used to create the graph. Transformer will send data consumed information to the cloud periodically. Data will get inserted to the mongodb. Datamart data consumption function will run to migrate data from MySQL to mongodb. Once the data is migrated graphs will be shown on DataSCAPE™ for transformer.

Transformer Health

The transformer health graph contains three lines. The first two lines show the oil temperature at the top and bottom of the transformer over time, and the third line shows the ambient temperature around the transformer over time.

Avg Usage Per Meter

The avg. kWh usage per meter graph shows seven lines. Each line will correspond to one of the days of the week plotted over 24 hours. Each point shows the avg usage for connected meters on the selected transformers over the chosen timeframe for the point selected. For example, selecting a three-week period and scrolling over the Wednesday line at the 03 point would be showing you the avg usage at 3am on Wednesday over the last 3 weeks for the meters connected to the selected transformers.

Network Loss

The network loss graph displays three lines. The first line shows the amount of energy sent out by the selected transformers to the connected meters. The second line shows how much was received by the meters, and the third shows the delta between these two.

Solar Return

The solar return graph shows two lines. The first plots the average kw returned by all the selected transformers over time and the second plots the avg ambient temperature around the devices.

Data Rate

Under Transformer Data Consumption page, we have Data Rate Button when user click on it will take it to Data Rate Page. Here user need to select the appropriate Circuit ID and HS/HH ID then click on procced button if the device is online it will fetch the Data Download and Upload rate.
