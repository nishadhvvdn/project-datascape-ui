Dashboard
The dashboard section is the first screen shown to a user and contains a high-level overview of the system. Here a user can explore the different maps and information on the system as a whole.
Mapping
The mapping section allows a user to view the network coverage, non-technical loss, outages, and other basic information of the devices on the system.
Graphing
The graphing section allows a user to visualize the data associated with the DTCs, transformers, and meters.
Reports
The reports section allows a user to create a number of on demand reports ranging from a report on outage times to network statistics and health of devices on the network.
Settings
The settings section allows a user to change their account's personal settings.
