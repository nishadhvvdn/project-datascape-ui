Opening the map

To open the map to its full screen size simply scroll over any of the DTCs (you will see a small annotation box with the DTC ID appear when the cursor is correctly positioned over one of the DTC icons) and click the icon. The screen will expand to fill the entire window instead of its original dashboard size.

Navigation Tools

Every map will display a small vertical bar in the top left corner of the map while you have the cursor over the map. From the top, down, the toolbar allows you to search for a specific city, zoom in and out, return to the original display before you started moving around the map, and drag/select.

Color Coding

Each icon will be colored either red or green. A green icon indicates the device is not experiencing any network related errors, while a red icon indicates that the device is experiencing network errors. These two icons are shown below.

DTC Control

To modify what DTCs are shown on the dashboard simply click the "Select Distribution Transformer Circuit" dropdown and select the DTCs you wish to be seen.
