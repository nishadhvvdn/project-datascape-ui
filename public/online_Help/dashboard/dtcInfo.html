Modifying the DTC Info

To modify what DTCs are included when calculating the numbers used for the Distribution Transformer Circuit Info dashboard simply click the "Select Distribution Transformer Circuit" dropdown and select the DTCs you wish to be used for the calculations.

Reading the DTC info

The DTC info dashboard shows a myriad of stats related to the DTCs in the system. These are used to give you a high-level overview of the system's current health. The dashboard contains ten unique displays which are each described in the following list.

  • 1. 24 hrs Total Power Flow (kW): This box shows the total amount of energy in the last 24 hours that has flown through the DTCs to the downstream devices.
  • 2. 24 hrs Avg. Power Factor: This box shows the average power factor in the last 24 hours for the displayed DTCs.
  • 3. 24 hrs Avg. Current (A): This box shows the average current in the last 24 hours for the displayed DTCs.
  • 4. 24 hrs Avg. Voltage (V): This box shows the average voltage in the last 24 hours for the displayed DTCs.
  • 5. Non-Technical Loss this Month: This box shows the total amount of loss in the currency you have set for the month.
  • 6. Transformers: This box shows the number of transformers connected to the displayed DTCs.
  • 7. Meters: This box shows the number of meters connected to the displayed DTCs.
  • 8. EV Chargers: This box shows the number of electric vehicle chargers connected to the displayed DTCs.
  • 9. Return (kW): This box shows the total amount of energy returned by each displayed DTC.
  • 10. Errors/Warnings: This box shows the total number of errors or warnings being reported by the DTCs.

EV Chargers List

When the EV Chargers box is clicked, you will be taken to a detailed list. This list will display all meters that are registered as an EV charger with the total amount of load over a specified period of time. The time period can be twenty-four hours, seven days, or thirty days depending on which option you select from the radio buttons at the bottom left of the screen will also display the total number of EV chargers and the total load on all the displayed chargers.

KW Return List

When the KW Return box is clicked, you will be taken to a detailed list. The list will display all meters with return, along with the amount of return over a specified period of time. The time period can be twenty-four hours, seven days, or thirty days depending on which option you select from the radio buttons at the bottom middle of the screen. The page will also display the total return.

Errors/Warnings List

When the errors/warnings box is clicked, you will be taken to a detailed list. The list will display all devices reporting errors or warnings. This list contains three filter dropdowns: one for DTC selection, another for transformers, and a final one for filtering by meter.

The list itself has six columns, as described in the following list:

  • 1. DTC ID: This column shows the ID of the DTC where the alarm is being reported.
  • 2. Transformer ID: This column shows the ID of the transformer where the alarm is being reported.
  • 3. HyperSPROUT™ ID: This column shows the ID of the HyperSPROUT™ where the alarm is being reported.
  • 4. Meter ID: This column shows the ID of the meter where the alarm is being reported. Note, this will say HyperSPROUT™ alarm if the alarm is specifically coming from a HyperSPROUT™ and no meters are involved.
  • 5. DB Timestamp: A timestamp will be shown here denoting when the alarm or warning was received.
  • 6. Alarm Type: The type of alarm (IE outage, tampering, battery failure, etc.) will be shown here.

Phase Angle

The phase is defined as the position of the waveform at a fraction of time period. Phase is expressed in angle or radian. Phase can also be an expression of relative displacement between two corresponding features (for example, peaks or zero crossings) of two waveforms having the same frequency. For the phase angle functionality in the DataSCAPE™ user needs to click on the power factor on the dashboard. It will take the User to the page where we can see the phasor graphs page.

Phasor Graph

On the phasor graphs page we have the DTC and transformer filter users need to select a particular DTC for which they want to see the transformer phasor graph. Once the fields are selected the graph will be displayed.

Sine Wave Graph

To see the sign wave graph, users need to select the graph type as Sine Wave. According to the filters selected it will plot the sine wave graph.
