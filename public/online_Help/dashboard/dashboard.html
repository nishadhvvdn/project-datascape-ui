The DTC info dashboard allows a user to see a snapshot of the system's current state.
Summary Map
The summary map provides the user with a graphical representation of the devices in the system and a summary on the current device state.
Outage Map
The outage map provides the user with a graphical representation of the devices in the system and current or historical outages they experienced.
Non-Technical Loss Map
The non-technical loss map provides the user with a graphical representation of the devices in the system and identifies likely points of theft.
Network Coverage Map
The network coverage map provides the user with a graphical representation of the devices in the system and their Wi-Fi reach.
