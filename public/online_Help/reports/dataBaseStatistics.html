Creating a Report

To create a report, begin by choosing a timeframe by clicking on the time picker, choosing a date, an hour, and the minute. Then, after filling this in, click the "Get Data" button. The report will instantly begin to load.

Reading the Report

The database statistics report shows the inserts, updates, commands and transactions per second the database experienced over the selected time, as well as any errors during that time period. The report contains seven columns.

  • 1. Inserts: The number of inserts that were performed on the database over the specified period of time
  • 2. Updates: The number of updates that were performed on the database over the specified period of time
  • 3. Commands: The number of commands that were performed on the database over the specified period of time
  • 4. Transactions Per Second: The avg number of transactions performed per second on the database over the specified period of time
  • 5. Error Type: The type of error that occured
  • 6. Error Description: A description of the error type
  • 7. Error Timestamp: A timestamp showing when the error happened

If the report has a large amount of data, it might separate into pages. In this case, you can choose how many entries to display, or which page to show, from the pagination tool bar in the bottom left of the report.

Sorting the Report

To sort the report, click the header of whichever column you would like to sort by. For instance, clicking on the error type header will switch the column between the default display, ascending, and descending.

Exporting a Report

The report gives four different options for exporting. You may export either all the data in the report or just the currently displayed data, in either a CSV format or a PDF format. To do this simply click on the export button in the top right corner of the report after filtering it for the desired data.
