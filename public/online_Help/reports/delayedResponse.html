Delayed Response

Creating a Report Reading the Report Sorting the Report Exporting the Report

Creating a Report

To create a report simply click the "Create Report" button.

Reading the Report

The delayed response report shows each device that is currently experiencing communication issues, and how many times it missed talking to the system when data was requested from it. The report contains three columns.

  • 1. Device ID: The serial number of the device
  • 2. Time Sent: The last time the device communicated with the system
  • 3. Data Intervals Delayed: The number of communication attempts the device has missed

If the report has a large amount of data, it might separate into pages. In this case, you can choose how many entries to display, or which page to show, from the pagination tool bar in the bottom left of the report.

Sorting the Report

To sort the report, click the header of whichever column you would like to sort by. For instance, clicking on the device ID header will switch the column between the default display, ascending, and descending.

Exporting a Report

The report gives four different options for exporting. You may export either all the data in the report or just the currently displayed data, in either a CSV format or a PDF format. To do this, simply click on the export button in the top right corner of the report after filtering it for the desired data.
