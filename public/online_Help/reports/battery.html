Creating a Report

To create a report simply click the "Create Report" button.

Reading the Report

The battery report shows the time used and left on the HyperSPROUT™ batteries, along with the battery voltage. The report contains four columns.

  • 1. HyperSPROUT™ Serial Number: The serial number of the HyperSPROUT™ the battery is in
  • 2. Battery Voltage: The current battery voltage
  • 3. Time Used on Battery: The number of minutes the battery has been used
  • 4. Time Left on Battery: The number of minutes left on the battery

If the report has a large amount of data, it might separate into pages. In this case, you can choose how many entries to display, or which page to show, from the pagination tool bar in the bottom left of the report.

Sorting the Report

To sort the report, click the header of whichever column you would like to sort by. For instance, clicking on the battery voltage header will switch the column between the default display, ascending, and descending.

Exporting a Report

The report gives four different options for exporting. You may export either all the data in the report or just the currently displayed data, in either a CSV format or a PDF format. To do this, simply click on the export button in the top right corner of the report after filtering it for the desired data.
