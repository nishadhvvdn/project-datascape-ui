Creating a Report

To create a report, begin by choosing a timeframe by clicking on the time picker, choosing a date, an hour, and the minute. Then, after filling this in, click the "Create Report" button. The report will instantly begin to load.

Reading the Report

The outage report shows the outages experienced over the specified time period. The report contains six columns.

  • 1. Event ID:The ID for the outage
  • 2. DTC ID: The ID of the DTC where the transformer is grouped
  • 3. Device ID: The ID of the transformer affected by outage
  • 4. Start Time: The start time of the outage
  • 5. End Time: The end time of the outage
  • 6. Duration: How long the outage lasted

If the report has a large amount of data, it might separate into pages. In this case, you can choose how many entries to display, or which page to show, from the pagination tool bar in the bottom left of the report.

Sorting the Report

To sort the report, click the header of whichever column you would like to sort by. For instance, clicking on the DTC ID header will switch the column between the default display, ascending, and descending.

Filtering the Report

To filter out the report for only a specified DTC, just type the DTC's ID in the text box above the right corner of the list labeled "Search by DTC ID". You can do the same with the Device ID filter next to the DTC filter.

Exporting a Report

The report gives four different options for exporting. You may export either all the data in the report or just the currently displayed data, in either a CSV format or a PDF format. To do this, simply click on the export button in the top right corner of the report after filtering it for the desired data.
