Creating a Report

To create a cloud statistics report, simply click the cloud statistics menu bar and a current report using the default Azure Region will be created. This will tell you the health of the Azure services used in that region. To change the region displayed just click the "Azure Region" dropdown and change it to the desired region.

Reading the Report
