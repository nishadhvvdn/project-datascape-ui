Creating a Report

To create a report simply click the "Create Report" button.

Reading the Report

The network statistics report shows general network-related data on the connected transformers and meters. This report shows the connected devices, managerial data load (data being used by the system; this does not include data used by phones and other network devices connected to the system), and connection status of the device. The report contains five columns.

  • 1. HyperSPROUT™ Serial Number: The serial number of the HyperSPROUT™ that is being displayed
  • 2. Total Meters: The number of meters connected to the particular transformer
  • 3. Connected Meters: The number of meters connected to the particular transformer that are communicating currently
  • 4. 24h Managerial Data Load (Bytes): The number of Bytes passed through the HyperSPROUT™ or meter by the system
  • 5. Status: The current communication status of the device (i.e. is it connected, disconnected, or experiencing a commincations fault)
  • 6. Meters List: This column contains a button that will change the list to display all of the meters connected to the transformer
  • 7. Serial Number: The serial number of the meter
  • 8. Meter Device ID: The device ID of the meter

If the report has a large amount of data, it might separate into pages. In this case, you can choose how many entries to display, or which page to show, from the pagination tool bar in the bottom left of the report.

Navigating the Report

To navigate back and forth in the report, simply click the button in the Meters List column to display all meters that are connected to the transformer that's serial number was displayed in the same row as the button you clicked. To return to the previous list click the blue HyperSPROUT™ link that has appeared above the top left corner of the list.

Sorting the Report

To sort the report, click the header of whichever column you would like to sort by. For instance, clicking on the total meters header will switch the column between the default display, ascending, and descending.

Filtering the Report

To filter out for only specified connection types, make sure you are on the meter page, then click the dropdown above the top right of the list and select the desired connection type to filter out for.

Exporting a Report

The report gives four different options for exporting. You may export either all the data in the report or just the currently displayed data, in either a CSV format or a PDF format. To do this, simply click on the export button in the top right corner of the report after filtering it for the desired data.
