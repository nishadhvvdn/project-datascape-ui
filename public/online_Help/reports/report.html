Reports

This page has five icons corresponding to the child dashboard pages in the DataSCAPEâ„¢ help screens. Each section has the respective title along with a small description of the section. The content of each section follows and corresponds with the above screenshot's red bubbled numbers:

Billing
The billing report creates KW usage reports from corresponding input files.
Cloud Statistics
The cloud statistics report gives snapshot of the current health of the Azure resources being used to run the cloud portion of the system.
Database Statistics
The database statistics report shows the activity and errors on the database over a specified period of time.
Non-Technical Loss
The non-technical loss report shows devices reporting loss within a specified period of time.
Network Response
The network response report shows the response rate of devices over a specified period of time.
Outages
The outage report shows the duration and start time of outages over a specified period of time.
Delayed Response
The delayed response report shows any currently unresponsive devices and how many intervals they have missed responding to.
Device Error
The device error report shows any errors reported by devices over a specified period of time.
Solar
The solar report shows the average and total return from the devices over a specified time.
Battery
The battery report shows the current status of the backup batteries on the devices.
Network Statistics
The network statistics report shows the connection status and managerial data load of devices on the network.
Data Rate
The Data rate report shows the information of the date rate for that time period.
