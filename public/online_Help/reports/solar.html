Creating a Report

To create a report, begin by choosing a timeframe by clicking on the time picker, choosing a date, an hour, and the minute. Then, after filling this in, click the "Create Report" button. The report will instantly begin to load.

Reading the Report

The solar report shows the amount of total and average return over one and thirty days. The report contains five columns.

  • 1. Serial Number: The serial number of the device with energy return
  • 2. Avg. 24h Return (KW): The average KW return from the device over the last twenty-four hours
  • 3. Total 24h Return (KW): The total KW return from the device over the last twenty-four hours
  • 4. Avg. 30 Day Return (KW): The average KW return from the device over the last thirty days
  • 5. Total 30 Day Return (KW): The total KW return from the device over the last thirty days

If the report has a large amount of data, it might separate into pages. In this case, you can choose how many entries to display, or which page to show, from the pagination tool bar in the bottom left of the report.

Sorting the Report

To sort the report, click the header of whichever column you would like to sort by. For instance, clicking on the serial number header will switch the column between the default display, ascending, and descending.

Filtering a Report

To filter out the report for only a specified serial number, just type the serial number in the text box above the right corner of the list labeled "Search by Serial Number".

Exporting a Report

The report gives four different options for exporting. You may export either all the data in the report or just the currently displayed data, in either a CSV format or a PDF format. To do this, simply click on the export button in the top right corner of the report after filtering it for the desired data.
